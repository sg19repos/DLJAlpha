# DLJ Alpha build in material ui and react-redux

## Features

-   React 16
-   Webpack 4
-   Babel 7
-   Hot Module Replacement
-   Internationalization
-   Redux
-   Daily Silver price chart - buying & selling price

## Installation

-   cd sankeycharts
-   npm install
-   npm start
-   visit `http://localhost:8080/`
